## 简介

UEditor是由百度web前端研发部开发所见即所得富文本web编辑器，具有轻量，可定制，注重用户体验等特点，开源基于MIT协议，允许自由使用和修改代码...
由此，小弟就不客气地添加一下功能：UEditor编辑器上传图片添加水印
